# Data Sources

GeoBeat aggregates data from multiple sources to provide comprehensive network analysis.

## Bitcoin

- **Bitnodes.io** - Real-time Bitcoin node tracking with geographic data

## Ethereum

- **Etherscan Node Tracker** - Ethereum node distribution by country
- **Rated Network** - Validator performance and geographic distribution
- **Miga Labs** - Consensus layer client diversity and infrastructure

## Filecoin

- **ProbeLab** - IPFS and Filecoin network measurements
- **Filstats** - Real-time Filecoin node telemetry

## Full Inventory

For a complete catalog of all data sources, see our [Data Source Inventory](https://github.com/DecentralizedGeo/geobeat/blob/main/data-sources/INVENTORY.md).
