# Introduction

Welcome to GeoBeat - a geographic decentralization index and dashboard for blockchain networks.

## What is GeoBeat?

GeoBeat measures physical decentralization across blockchain networks by analyzing the geographic distribution of nodes and validators. Our goal is to provide transparent, data-driven insights into network resilience and censorship resistance.
